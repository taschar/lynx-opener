# Lynx Opener

[English version](#english) | [Ελληνική έκδοση](#greek)

## English

Lynx Opener is a Chrome-based browser extension for Linux that allows you to open any link in the **Lynx** text-browser within your favorite terminal emulator via a simple right-click.

### Features

- **Context Menu Integration:** Right-click any link to see the "Open link in Lynx" option.
- **Customizable Terminal:** Choose your preferred terminal (gnome-terminal, alacritty, kitty, xterm, etc.).
- **Fullscreen Toggle:** Option to launch the terminal in fullscreen mode or windowed mode.
- **Lightweight & Fast:** Uses Native Messaging to bridge the browser with your OS.
- **Modern UI:** Clean and beautiful settings page, now supporting multiple languages (English/Greek).

### Requirements

To use this extension, you must have the following installed on your Linux system:

1.  **Lynx Browser:** `sudo apt install lynx` (or your distro's equivalent).
2.  **Python 3:** Required for the native messaging host.
3.  **Terminal Emulator:** The terminal you want to use.

### Installation

#### 1. Load the Extension
1.  Open your browser and navigate to `chrome://extensions/`.
2.  Enable **Developer mode** (top right corner).
3.  Click **Load unpacked**.
4.  Select the `extension` folder from this repository.
5.  **Important:** Copy the **Extension ID** generated by the browser (e.g., `obonmkgfcjhloc...`).

#### 2. Install the Native Messaging Host
1.  Open a terminal in the root of this repository.
2.  Run the installation script:
    ```bash
    chmod +x install.sh
    ./install.sh
    ```
3.  When prompted, paste the **Extension ID** you copied in the previous step.
4.  The script will register the host for Chrome, Chromium, Brave, and Edge.

### Configuration & Settings

#### Accessing Settings
To customize the extension:
1.  Click the **puzzle piece icon** (Extensions) in your browser toolbar.
2.  Find **Lynx Opener** and click the three dots (More actions).
3.  Select **Options**. Alternatively, go to `chrome://extensions/`, find Lynx Opener, and click **Details** -> **Extension options**.

#### Available Options
- **Terminal Command:** 
  - **Default:** `x-terminal-emulator` (This points to your system's default terminal in most Debian-based distros like Ubuntu).
  - You can change this to any installed terminal command (e.g., `gnome-terminal`, `kitty`, `alacritty`, `konsole`).
- **Open in Fullscreen:**
  - A toggle to decide if the terminal should open in fullscreen mode.
  - **Default:** Enabled (On).

## Ελληνικά

Το Lynx Opener είναι μια επέκταση περιηγητή για Linux (Chrome, Chromium κ.α.) που σας επιτρέπει να ανοίγετε οποιονδήποτε σύνδεσμο στον περιηγητή κειμένου **Lynx** μέσα στον αγαπημένο σας εξομοιωτή τερματικού με ένα απλό δεξί κλικ.

### Χαρακτηριστικά

- **Ενσωμάτωση στο Μενού Περιβάλλοντος:** Δεξί κλικ σε οποιονδήποτε σύνδεσμο για την επιλογή "Open link in Lynx".
- **Προσαρμόσιμο Τερματικό:** Επιλέξτε το τερματικό που προτιμάτε (gnome-terminal, alacritty, kitty, xterm κ.λπ.).
- **Λειτουργία Πλήρους Οθόνης:** Επιλογή για εκκίνηση του τερματικού σε πλήρη οθόνη ή σε παράθυρο.
- **Ελαφρύ & Γρήγορο:** Χρησιμοποιεί το Native Messaging για τη σύνδεση του περιηγητή με το λειτουργικό σας σύστημα.
- **Σύγχρονο UI:** Καθαρή και όμορφη σελίδα ρυθμίσεων, πλέον με υποστήριξη πολλαπλών γλωσσών (Αγγλικά/Ελληνικά).

### Απαιτήσεις

Για να χρησιμοποιήσετε αυτήν την επέκταση, πρέπει να έχετε εγκατεστημένα τα εξής στο Linux σύστημά σας:

1.  **Lynx Browser:** `sudo apt install lynx` (ή το αντίστοιχο της διανομής σας).
2.  **Python 3:** Απαιτείται για τον native messaging host.
3.  **Εξομοιωτής Τερματικού:** Το τερματικό που θέλετε να χρησιμοποιήσετε.

### Εγκατάσταση

#### 1. Φόρτωση της Επέκτασης
1.  Ανοίξτε τον περιηγητή σας και μεταβείτε στο `chrome://extensions/`.
2.  Ενεργοποιήστε τη **Λειτουργία προγραμματιστή** (πάνω δεξιά γωνία).
3.  Κάντε κλικ στο **Φόρτωση μη συσκευασμένης επέκτασης**.
4.  Επιλέξτε το φάκελο `extension` από αυτό το αποθετήριο.
5.  **Σημαντικό:** Αντιγράψτε το **Αναγνωριστικό επέκτασης (Extension ID)** που δημιουργήθηκε (π.χ., `obonmkgfcjhloc...`).

#### 2. Εγκατάσταση του Native Messaging Host
1.  Ανοίξτε ένα τερματικό στη ρίζα αυτού του αποθετηρίου.
2.  Εκτελέστε το σενάριο εγκατάστασης:
    ```bash
    chmod +x install.sh
    ./install.sh
    ```
3.  Όταν σας ζητηθεί, επικολλήστε το **Extension ID** που αντιγράψατε στο προηγούμενο βήμα.
4.  Το σενάριο θα καταχωρήσει τον host για Chrome, Chromium, Brave και Edge.

### Ρυθμίσεις

#### Πρόσβαση στις Ρυθμίσεις
Για να προσαρμόσετε την επέκταση:
1.  Κάντε κλικ στο **εικονίδιο του παζλ** (Επεκτάσεις) στη γραμμή εργαλείων του περιηγητή σας.
2.  Βρείτε το **Lynx Opener** και κάντε κλικ στις τρεις τελείες (Περισσότερες ενέργειες).
3.  Επιλέξτε **Επιλογές**. Εναλλακτικά, μεταβείτε στο `chrome://extensions/`, βρείτε το Lynx Opener και κάντε κλικ στο **Λεπτομέρειες** -> **Επιλογές επέκτασης**.

#### Διαθέσιμες Επιλογές
- **Εντολή Τερματικού:** 
  - **Προεπιλογή:** `x-terminal-emulator` (Αυτό δείχνει στο προεπιλεγμένο τερματικό του συστήματός σας στις περισσότερες διανομές βασισμένες στο Debian, όπως το Ubuntu).
  - Μπορείτε να το αλλάξετε σε οποιαδήποτε εγκατεστημένη εντολή τερματικού (π.χ., `gnome-terminal`, `kitty`, `alacritty`, `konsole`).
- **Άνοιγμα σε Πλήρη Οθόνη:**
  - Επιλογή για το αν το τερματικό θα ανοίγει σε λειτουργία πλήρους οθόνης.
  - **Προεπιλογή:** Ενεργοποιημένο.

## Πώς Λειτουργεί

Η επέκταση χρησιμοποιεί το **Native Messaging API** του Chrome. 
- Το σενάριο `background.js` ανιχνεύει το κλικ στο μενού περιβάλλοντος.
- Στέλνει τη διεύθυνση URL, την προτίμηση τερματικού και την επιλογή πλήρους οθόνης σε ένα σενάριο Python (`host/lynx_opener.py`).
- Το σενάριο Python εκτελεί την εντολή για την εκκίνηση του τερματικού με το Lynx.
