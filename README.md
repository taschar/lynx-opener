# Lynx Opener

[English version](#english) | [Ελληνική έκδοση](#greek)

<a name="english"></a>
## English

Lynx Opener is a browser extension for Linux (Chrome, Firefox, and derivatives) that allows you to open any link in the **Lynx** text-browser within your favorite terminal emulator via a simple right-click.

### Features

- **Cross-Browser Support:** Works on Chrome, Chromium, Brave, Edge, and **Firefox**.
- **Context Menu Integration:** Right-click any link to see the "Open link in Lynx" option.
- **Customizable Terminal:** Choose your preferred terminal (gnome-terminal, alacritty, kitty, xterm, xfce4-terminal, etc.).
- **Fullscreen Toggle:** Option to launch the terminal in fullscreen mode or windowed mode.
- **Lightweight & Fast:** Uses Native Messaging to bridge the browser with your OS.
- **Modern UI:** Clean and beautiful settings page, supporting 16+ languages.

### Requirements

To use this extension, you must have the following installed on your Linux system:

1.  **Lynx Browser:** `sudo apt install lynx` (or your distro's equivalent).
2.  **Python 3:** Required for the native messaging host.
3.  **Terminal Emulator:** The terminal you want to use.

### Installation

#### 1. Load the Extension

**For Chrome/Chromium/Brave/Edge:**
1.  Open your browser and navigate to `chrome://extensions/`.
2.  Enable **Developer mode** (top right corner).
3.  Click **Load unpacked**.
4.  Select the `extension` folder from this repository.
5.  **Important:** Copy the **Extension ID** generated by the browser.

**For Firefox:**
1.  Open Firefox and navigate to `about:debugging#/runtime/this-firefox`.
2.  Click **Load Temporary Add-on...**.
3.  Select the `manifest.firefox.json` file inside the `extension` folder. (Firefox allows selecting any json file as manifest).
4.  Note: Temporary add-ons in Firefox are removed when you restart the browser. For permanent use, you would need to sign the extension or use Firefox Developer Edition/Nightly.

#### 2. Install the Native Messaging Host
1.  Open a terminal in the root of this repository.
2.  Run the installation script:
    ```bash
    chmod +x install.sh
    ./install.sh
    ```
3.  Follow the prompts:
    - Paste the **Extension ID** (for Chrome-based browsers).
    - If you only use Firefox, you can leave the Chrome ID blank.
4.  The script will register the host for all supported browsers.

### Configuration & Settings

#### Accessing Settings
To customize the extension:
1.  Click the **Extensions icon** in your browser toolbar.
2.  Find **Lynx Opener** and go to **Options** or **Manage Extension** -> **Options**.

#### Available Options
- **Terminal Command:** 
  - **Default:** `x-terminal-emulator` (points to your system's default terminal).
  - You can change this to any installed terminal (e.g., `gnome-terminal`, `kitty`, `xfce4-terminal`).
- **Open in Fullscreen:**
  - Toggle for fullscreen mode.

## Ελληνικά

Το Lynx Opener είναι μια επέκταση περιηγητή για Linux (Chrome, Firefox και παράγωγα) που σας επιτρέπει να ανοίγετε οποιονδήποτε σύνδεσμο στον περιηγητή κειμένου **Lynx** μέσα στον αγαπημένο σας εξομοιωτή τερματικού με ένα απλό δεξί κλικ.

### Χαρακτηριστικά

- **Υποστήριξη Cross-Browser:** Λειτουργεί σε Chrome, Chromium, Brave, Edge και **Firefox**.
- **Ενσωμάτωση στο Μενού Περιβάλλοντος:** Δεξί κλικ σε οποιονδήποτε σύνδεσμο για την επιλογή "Open link in Lynx".
- **Προσαρμόσιμο Τερματικό:** Επιλέξτε το τερματικό που προτιμάτε (gnome-terminal, alacritty, kitty, xterm, xfce4-terminal κ.λπ.).
- **Λειτουργία Πλήρους Οθόνης:** Επιλογή για εκκίνηση του τερματικού σε πλήρη οθόνη ή σε παράθυρο.
- **Ελαφρύ & Γρήγορο:** Χρησιμοποιεί το Native Messaging για τη σύνδεση του περιηγητή με το λειτουργικό σας σύστημα.
- **Σύγχρονο UI:** Καθαρή και όμορφη σελίδα ρυθμίσεων, με υποστήριξη 16+ γλωσσών.

### Απαιτήσεις

Για να χρησιμοποιήσετε αυτήν την επέκταση, πρέπει να έχετε εγκατεστημένα τα εξής στο Linux σύστημά σας:

1.  **Lynx Browser:** `sudo apt install lynx` (ή το αντίστοιχο της διανομής σας).
2.  **Python 3:** Απαιτείται για τον native messaging host.
3.  **Εξομοιωτής Τερματικού:** Το τερματικό που θέλετε να χρησιμοποιήσετε.

### Εγκατάσταση

#### 1. Φόρτωση της Επέκτασης

**Για Chrome/Chromium/Brave/Edge:**
1.  Ανοίξτε τον περιηγητή σας και μεταβείτε στο `chrome://extensions/`.
2.  Ενεργοποιήστε τη **Λειτουργία προγραμματιστή** (πάνω δεξιά γωνία).
3.  Κάντε κλικ στο **Φόρτωση μη συσκευασμένης επέκτασης**.
4.  Επιλέξτε το φάκελο `extension` από αυτό το αποθετήριο.
5.  **Σημαντικό:** Αντιγράψτε το **Αναγνωριστικό επέκτασης (Extension ID)** που δημιουργήθηκε.

**Για Firefox:**
1.  Ανοίξτε τον Firefox και μεταβείτε στο `about:debugging#/runtime/this-firefox`.
2.  Κάντε κλικ στο **Load Temporary Add-on...**.
3.  Επιλέξτε το αρχείο `manifest.firefox.json` μέσα στο φάκελο `extension`.
4.  Σημείωση: Οι προσωρινές επεκτάσεις στον Firefox αφαιρούνται όταν κάνετε επανεκκίνηση. Για μόνιμη χρήση, η επέκταση πρέπει να υπογραφεί ή να χρησιμοποιηθεί η έκδοση Firefox Developer Edition/Nightly.

#### 2. Εγκατάσταση του Native Messaging Host
1.  Ανοίξτε ένα τερματικό στη ρίζα αυτού του αποθετηρίου.
2.  Εκτελέστε το σενάριο εγκατάστασης:
    ```bash
    chmod +x install.sh
    ./install.sh
    ```
3.  Ακολουθήστε τις οδηγίες:
    - Επικολλήστε το **Extension ID** (για Chrome-based browsers).
    - Αν χρησιμοποιείτε μόνο Firefox, μπορείτε να αφήσετε το πεδίο κενό.
4.  Το σενάριο θα καταχωρήσει τον host για όλους τους υποστηριζόμενους περιηγητές.

### Ρυθμίσεις

#### Πρόσβαση στις Ρυθμίσεις
Για να προσαρμόσετε την επέκταση:
1.  Κάντε κλικ στο **εικονίδιο των επεκτάσεων** στη γραμμή εργαλείων.
2.  Βρείτε το **Lynx Opener** και μεταβείτε στις **Επιλογές**.
